Change Password
Your password is stored as a Vercel Environment Variable ‚Äî it never appears in your code or GitHub repo.

üîë How to change your password

  1. Go to vercel.com ‚Üí open your portfolio project
  2. Click Settings ‚Üí Environment Variables
  3. Find ADMIN_PASSWORD and click the edit (pencil) icon
  4. Enter your new password value and click Save
  5. Go to the Deployments tab ‚Üí click the three dots on the latest deploy ‚Üí Redeploy
  6. Done ‚Äî your new password is live on every device within ~30 seconds.

üõ°Ô∏è First time setup

  1. Go to Vercel ‚Üí your project ‚Üí Settings ‚Üí Environment Variables
  2. Click Add New
  3. Name: ADMIN_PASSWORD  |  Value: your chosen password
  4. Make sure all three environments are checked: Production, Preview, Development
  5. Click Save, then redeploy.
How to Deploy Updates
When you click "Save & Deploy", your changes are saved to content.json. To go live you need to push to GitHub ‚Äî Vercel will auto-deploy.

üöÄ Deployment workflow

  1. Click Save & Deploy above ‚Äî this downloads an updated content.json to your computer.
  2. Replace the content.json file in your project folder with the downloaded one.
  3. Open your terminal in the project folder and run: git add content.json && git commit -m "Update content" && git push
  4. Vercel detects the push and automatically redeploys your site ‚Äî usually live in under 30 seconds.

üìÅ Your project folder structure

  1. index.html ‚Äî your main portfolio page
  2. admin/index.html ‚Äî this admin panel
  3. content.json ‚Äî all your site content (edit this via admin)
